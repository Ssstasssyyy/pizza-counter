# Pizza Counter

Веб-приложение для автоматического подсчёта приготовленных пицц на изображениях с использованием искусственного интеллекта.

## Описание проекта

Приложение позволяет:

- загружать изображение;
- автоматически находить пиццы с помощью модели YOLOv8;
- отображать количество найденных пицц;
- сохранять историю обработок в базе данных SQLite;
- экспортировать историю в Excel.

Проект разработан на Python в рамках учебной практики по дисциплине «Системы искусственного интеллекта».

## Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Ssstasssyyy/pizza-counter.git
```

### 2. Перейти в папку проекта

```bash
cd pizza-counter
```

### 3. Создать виртуальное окружение

```bash
python -m venv venv
```

### 4. Активировать окружение

Windows

```bash
venv\Scripts\activate
```

macOS/Linux

```bash
source venv/bin/activate
```

### 5. Установить зависимости

```bash
pip install -r requirements.txt
```

### 6. Запустить приложение

```bash
python app.py
```

После запуска приложение будет доступно по адресу

```
http://127.0.0.1:5000
```

## Демонстрация работы

1. Загрузить изображение.

4. При необходимости скачать Excel-отчёт.
